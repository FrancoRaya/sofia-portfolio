Final Project: El Saladillo Viaduct

Project information

  • Category: Architecture
  • Project name: El Saladillo Viaduct: Park and Interactive Center
  • My tasks: Designer. Technical documentation and visualization.
  • Project URL: YouTube

This project aims to be a sensory illusion with a spatial paradox, where landscape and architecture merge: a reflection. A mirror where the viaduct lays it fundations, to rise and frame the sky between the arches. The mirror increases the value of things and magnifies them.
